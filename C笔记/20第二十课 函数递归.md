# 函数递归

### 函数内部调用自己

```c
void func(int n)
{
    printf("%d\n", n);
    func(n + 1);
}
int main()
{
    func(0);
}
```

若陷入这样的死循环，使用 ==ctrl+c== 结束程序

函数结束，使用==return==
